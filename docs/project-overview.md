# Project Overview

FOLD is a premium, mobile-first laundry pickup-and-delivery marketing site for Montgomery, Alabama. The homepage is designed to feel more like a logistics app than a traditional service brochure while preserving the brand voice around transparency, local focus, and convenience.

## Live Site

- [Homepage](https://wglewis0721.github.io/laundry-delivery/index.html)

## Core Brand Messages

- Drop the bags. Keep the weekend.
- Weighed by the bag transparency.
- SMS tracking at each milestone.
- Receipt-before-charge billing.
- Montgomery-first service positioning.

## Homepage Highlights

- Premium hero with app-style ZIP availability check
- Floating order-tracking card and lifestyle imagery
- Modern bento-grid explanation of the FOLD difference
- Interactive pricing recommendation calculator
- Emotional lifestyle section focused on reclaiming weekends
- Modern service-area checker with mobile-first layout
- Refined testimonials, FAQ copy, and closing CTA

## Technical Notes

- Built as a static HTML page with Tailwind CSS via CDN.
- Uses semantic HTML, responsive layout patterns, and scroll-reveal interactions.
- Keeps JavaScript lightweight for performance and accessibility.
- Placeholder images use picsum.photos until final photography is added.
